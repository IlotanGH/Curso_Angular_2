One Note ==> Programa para tomar notas y apuntes de Microsoft.

mklink /D node_modules ..\..\node_modules ==> linkar la carpeta node_modules por que pesa mucho para no tenerla repetida muchas veces.

ng new -si libros ==> crea un modulo "libros"

ng g s buscar ==> crea un servicio "buscar"

ng g c libros ==> crea un componente "libros"

npm start ==> arrancar el servidor

ng serve ==> arrancar el servidor

ng serve --preserve-symlinks ==> arrancar el servidor conservando los linkados simulados
